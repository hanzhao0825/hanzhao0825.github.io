Unity AR Demo Part1 - Using Vuforia

In this tutorial, we are going to set up a demo Unity AR project step by step

Get your Vuforia ready

  1. Goto “https://developer.vuforia.com/user/register” and create an account
  2. Goto “https://developer.vuforia.com/license-manager” and create a license key for your application.
  3. Goto “https://developer.vuforia.com/downloads/sdk” and download Vuforia SDK for Unity
  4. Open a new project in Unity
  5. Double-Click the package you just downloaded and import all the contents

Build the scene

  1. Delete the main camera and replace it with ARCamera in “Vuforia/Prefabs”
  2. Fill in your App License Key
  3. Create the package for the image target
    a. Goto “https://developer.vuforia.com/targetmanager/“ and add a database
    b. In the database, add a simple image target (you can use this), and set the width to “1”
    c. Download the database
    d. Double-Click it and import the database
    e. Under ARCamera, select “Load YOUR database” and “activate”
  4. Add the image target to your scene
    a. Drop the “image target” from Vuforia/Prefabs to the scene
    b. Choose the right database and target for your image target
  5. Add a sphere into the scene, and drag it onto the image target
  6. Adjust the scale and the position of the sphere for better look

Run the project in the editor

  1. Click the “Play” button, the camera will open
  2. Put the image before the camera, you will see that sphere
