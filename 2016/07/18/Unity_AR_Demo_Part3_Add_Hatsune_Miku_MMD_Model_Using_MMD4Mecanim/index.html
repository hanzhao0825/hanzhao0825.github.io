Unity AR Demo Part3 - Add Hatsune Miku MMD Model Using MMD4Mecanim


Install MMD4Mecanim

  1. Download MMD4Mecanim from wherever you can find it
  2. Import the content to Unity

Import Hatsune Miku MMD Model

  1. Download your favorite MMD model for Hatsune Miku (I am using Tda in this tutorial)
  2. Copy the entire folder to “\Assets”, unity will automatically import the content
  3. Find the FMX format model file
  4. In “Inspector”, click “Agree”, then click “Process” and wait for the conversion
  5. Replace the sphere with the new FBX format model
  6. Adjust the intensity of the light if needed

Build Setting Change for iOS (Very Important!!)

  1. Find “Player Settings -> Other Settings”
  2. Deselect “Auto Graphics API”
  3. Place OpenGLES2 before Metal

Enjoy
