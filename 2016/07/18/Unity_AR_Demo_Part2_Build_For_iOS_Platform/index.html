Unity AR Demo Part2 - Build For iOS Platform

Once everything looks fine in the editor, we can build the project for IOS platform

  1. In “File - Build Settings”, select iOS
  2. Open Player Settings
  3. Edit Company Name and Product Name
  4. Edit Bundle Identifier in “Other Settings”
  5. Click “Build and Run”
