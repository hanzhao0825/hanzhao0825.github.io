Fix Pokemon Go Parse Error When Installing APK

Check you android version. If your version is under 4.4 (mine is 4.2), go on reading this article and fix the “Parse Error”

Caution : Instructions below might be destructive for your android phone. Do it at your own risk :)

Root your device

  1. Use whatever app to get your device rooted

Install Xposed

  1. Note that the current version of Xposed only support android 5.0 and above, so you need to download and install Xposed 2.7
  2. Run Xposed, click Install/Update

Install ComlatibleUp

  1. I use ComlatibleUp 3.2, and I don’t know if other versions will work
  2. Run ComlatibleUp
  3. In Xposed, tick the box next to “ComlatibleUp”

Reboot and Enjoy!
