LeetCode 239. Sliding Window Maximum

Given an array nums, there is a sliding window of size k which is moving from the very left of the array to the very right. You can only see the k numbers in the window. Each time the sliding window moves right by one position.

For example,

Given nums = [1,3,-1,-3,5,3,6,7], and k = 3.

Therefore, return the max sliding window as [3,3,5,5,6,7].

Solution in linear time using deque

No need to save the entire window, just save the numbers that might be used later.

For example, if the current window is like: “1 3 2”. We can see that the “1” is redundant since the “3” behind it is bigger. So, “3 2” shall be enough for later use. The sequence we need to maintain should be descending at any time.

We use a deque. When a new number come into the deque, make sure the sequence should remain a descending one.

Time Complexity : O(n)

Space Complexity : O(k)
